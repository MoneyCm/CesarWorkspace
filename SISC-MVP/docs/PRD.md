# PRD - SISC MVP

Objetivo: MVP para la Secretaría de Seguridad y Convivencia de Jamundí.

Usuarios: Admin SISC, Analista Observatorio, Cargador de datos, Consulta interna, Público.

Casos de uso: registro fuentes, carga archivos, ver indicadores, mapas y generar boletines.

Alcance MVP: ingestión CSV, módulos A-F básicos, portal público con datos agregados.

No alcance: ML avanzado, microservicios completos, integraciones en producción.
