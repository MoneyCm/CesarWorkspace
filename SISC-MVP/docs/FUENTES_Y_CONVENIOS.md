# Plantilla Fuentes y Convenios

Campos sugeridos:
- `nombre`, `responsable`, `periodicidad`, `canal` (API/CSV/Excel), `campos` (JSON con mapeo)

Ejemplo:

```
nombre: Policía Metropolitana
responsable: Comandante (nombre)
periodicidad: semanal
canal: CSV vía FTP
campos: {fecha_hecho:date, hora:time, tipo_evento:text, ...}
```
