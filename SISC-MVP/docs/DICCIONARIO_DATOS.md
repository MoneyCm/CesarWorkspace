# Diccionario de datos (ejemplo)

- `fecha_hecho` (date): Fecha del evento.
- `hora` (time): Hora del hecho.
- `tipo_evento` (text): Clasificación principal (Hurto, Lesión, etc.).
- `modalidad` (text): Modalidad del evento.
- `barrio` (text): Unidad territorial.
- `geom` (geometry): Punto WGS84 (lat/lon).
